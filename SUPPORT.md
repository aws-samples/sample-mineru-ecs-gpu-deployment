# Support

## Getting Help

This is a sample code repository provided for demonstration and learning purposes. 

### GitHub Issues

For questions, bug reports, or feature requests related to this sample code, please use the [GitHub Issues](../../issues) tab.

When opening an issue, please include:
- A clear description of the problem or question
- Steps to reproduce (for bugs)
- Your environment details (AWS region, instance types, etc.)
- Relevant logs or error messages

### AWS Support

For issues related to AWS services themselves (not this sample code):
- **AWS Support Plans**: If you have an AWS Support plan, create a support case in the [AWS Support Center](https://console.aws.amazon.com/support/)
- **AWS Forums**: Visit [AWS re:Post](https://repost.aws/) for community support
- **AWS Documentation**: Check the [AWS Documentation](https://docs.aws.amazon.com/)

### MinerU Support

For issues specific to the MinerU library:
- Visit the [MinerU GitHub repository](https://github.com/opendatalab/MinerU)
- Check their documentation and issue tracker

## Important Notes

- This is **sample code** for demonstration purposes
- It is **not officially supported** as a product
- Community contributions and discussions are welcome via GitHub Issues
- For production deployments, please review and adapt the code to your specific requirements

## Security Issues

If you discover a potential security issue, please refer to our [Security Policy](SECURITY.md) for reporting instructions. **Do not** create a public GitHub issue for security vulnerabilities.
